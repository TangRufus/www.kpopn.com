噴完輕鬆一搓就能去角質？韓國女生推薦愛用的去角質神器是什麼？

通過
Iris
-
2018-03-25
  • 標籤
  • W.lab
廣告

在韓國的電視節目裡常常可以看到韓國人喜愛搓澡去角質的習慣，除了去付費澡堂洗澡的時候順便給專業的搓澡員搓澡，也會買一些去角質產品放在家裡自已用 (像 EXO 的伯賢就在《認識的哥哥》裡說過 D.O. 都會用磨砂膏幫他搓背～)。今天 Kpopn 要跟大家介紹一款顧客滿意度5分中得到4.9的高分、被韓國女生稱作去角質神器的 W.Lab One Shot 去角質噴霧！

像這樣先搖勻噴霧噴在皮膚上，再用手輕輕按摩，老廢角質就會結成一堆黑色的點點，達到去角質的效果，來看看韓國網友實際使用的照片：

難怪韓國女生會稱它為去角質神器！這也去得太輕鬆了吧！(立馬丟掉手中其他去角質產品 XD) 不過光靠這噴出來黑黑的水，再輕輕一搓為什麼就可以快速去除老化角質呢？快來看看 One Shot 去角質噴霧的三個大重點！

 Point 01：泥炭水與果酸

泥炭水是北歐人沐浴時愛用的洗劑，天然的成分具有吸附皮膚上老化廢物角質的功能，還有給予去角質過後的肌膚光彩的效果，是這款 One Shot 去角質噴霧最主要的成分，也是為什麼它噴出來是黑色的原因。而 AHA 果酸本來就很廣泛的運用於各種去角質產品中，可以讓老化、死去的角質細胞脫落，促進皮膚代謝，髒的角質去除掉後，不但能暢通毛孔，還能同時改善膚質與膚色。

 Point 02：後續肌膚鎮定

上述有提到這款噴霧有添加果酸，果酸是由多種天然蔬果中萃取的自然酸，在吸附掉老廢角質後，能同時給予鎮定的效果，不會讓皮膚感到乾澀不適，是很天然的去角質兼鎮定的成分。

 Point 03：兼具保濕效果

以添加神經醯胺及燕麥萃取物作為最後保濕功能的成分。神經醯胺是一種油溶性的天然保濕因子，能建立維持肌膚脂質的屏障，常常被拿來作為適合熟齡肌膚滋潤的高保濕成分。在臺灣，燕麥雖然比較少聽到運用於保養品中，但其實在國外已經被多數使用在針對敏感性肌膚研發的保養品之中，燕麥兼具溫和與安定的特性，卻同時含有超強保濕力，是一種低敏感高保濕的成分，兩種成分對於去角質後的受損肌膚都有很好的修護效果。

現在在韓國多數 W.Lab 有進駐的美妝店都有賣這款 One Shot 去角質噴霧，一罐大容量 300ml，售價是韓幣15,000元 (約台幣429元)。

從官網上滿分5分中拿到4.9分的評價，以及韓國網友們讚譽有加的使用後記來看，難怪這款 One Shot 去角質噴霧會被冠上去角質神器的名號，小編我也是看得很想要買一罐來用用看啊！
